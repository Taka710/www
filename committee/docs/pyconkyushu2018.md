# PyConKyushu 2018 in Fukuoka

## 概要  
Pythonをテーマとした九州で初めてのカンファレンスを福岡で開催。
九州という地域の序章という意味を込めて、テーマを「Prologue」としました。
  
当日は、スタッフも含めて180名以上の方に参加いただき、成功を収めました。

https://kyushu.pycon.jp/2018/
